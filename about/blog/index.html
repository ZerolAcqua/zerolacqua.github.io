关于博客

博客搭建

  • 博客基于 Hexo 框架，使用 Butterfly 主题，部署于 腾讯云轻量应用服务器，并在 Github Page 上有一份镜像。

  • 图片、CSS、JS 等静态资源使用 腾讯云 CDN 加速。

  • 图片压缩使用 TinyPNG API 完成。

  • 部分图片由 路过图床 托管。

  • 朋友圈 部署于 Github + Vercel 。

  • 评论系统 Waline 服务端与数据库部署于腾讯云轻量应用服务器

  • AI 摘要服务由 洪墨摘要AI 提供

  • Waline 部署于 Vercel，并由 LeanCloud 托管数据。

  • github-calendar api 部署于 Vercel。

  • github-readme-stats api 部署于 Vercel。

  • 参考使用了众多博客大佬的方法与插件。
