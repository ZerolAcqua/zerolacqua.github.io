找实习的心路经历

发表于2025-05-25|更新于2025-07-18|随笔
|总字数:4.3k|阅读时长:12分钟|浏览量:

没有什么准备，我孤身来到了这座熟悉而陌生的城市——上海。现在我坐在这个仄逼的出租房里，回顾我找实习的心路历程。

找实习期间也是随手记了一些感受吧，这里就放出来吧。

记录

时间

2024-09-01

开始刷 Leetcode 但是整体是一个比较轻松的状态，期间也就是做了做课题组的快拼项目，准备投会议的。而且中间还在做自己的小项目（前端的甚至），当时还没有想好到底要准备什么岗位。

2024-11-30

决定走 Java 后端，加了小傅哥的星球，开始做项目，选了小型支付商城。刷题的强度也慢慢上来了。

2025-01-18

基本才把小型支付商城写完，比预期的慢太多了。

2025-02-07

寒假在家效率很低，初十返校之后开始全力看八股。

2025-02-16

怀着非常忐忑的心情去拍了简历照，结果比我想象的要简单，照相馆的小猫咪还蛮可爱的。（期间十一点半的时候摸摸鱼，给自己喜欢的项目水了些 PR）

2025-02-20

组会上导师让我再把现在这个会议投了，再写篇正刊，我心态有点崩。组会最后和导师说准备找实习，导师爽快的答应了。

2025-02-26

腾讯的提前实习招聘一直不敢投，觉得自己去了就是白给。各厂的暑期实习也陆续开了，很焦虑。中间也是磨磨叽叽整出了个简历。

2025-03-02

准备开始投简历，发现投简历要投的信息也太多了。但是八股还有计算机系统的没还看完。JVM、多线程、Redis、SQL 感觉又忘完了。刷的算法题又有不会的。

2025-03-04

第一次做测评，还真是一股子的行测味。然后它的心理测试相关的我感觉也好强的针对性x

2025-03-05

上午试着看了下 Openai 代码评审的组件项目，感觉不难，但是纠结还要不要花时间加这个项目了，要不简历先投了吧。

刷了下牛客网上的真题，很焦虑，感觉考得很多我都不会（指不全是算法题，有选项题的那种）。算法题没那么难了，但是第一次做 ACM 模式的算法题光处理输入输出就有够麻烦了，很焦虑……打算之后把真题做一点，然后补强 java 基础题，每天继续看看八股 （）

2025-03-06

腾讯的测评不是北森的是自家的系统，不少职场情景题……我草怎么上午做完测评下午就约面了啊，我好慌啊……好慌好慌好慌

2025-03-07

心情莫名其妙地平静下来了，大概绝望到头反而就会淡定了

不是哥们，美团的笔试也太早了吧？？明天就笔试？？还 tm 有 AI 面试啊？

2025-03-08

试了一下 AI 面试，题目都好难，一点信心也没有。
晚上的笔试也做的一塌糊涂……

活过 AI 面试了，至少对面试的恐惧感没那么强烈了……

2025-03-10

腾讯的面试延到明天了，我还是觉得根本准备不完

2025-03-11

并没有我想象的那么难，面试官还是挺友好的，八股自我感觉也答的不错，但是手撕代码十分钟写快排还是不行（

2025-03-14

饿了吗的笔试，依然是操作系统和设计模式是短板。总之加油吧，这次的算法题比之前有进步多了，三道 A 了两道题

2025-03-15

OPPO 的笔试，选项题里有 C++ 和 Pyhton，给我整无语了。算法题感觉不难，但是我做不出来。第二题只 A 了 0.06 （）

2025-03-16

阿里云笔试，算法题总共只 A 了 0.05。听说第三题比第一题简单（）

2025-03-17

收到阿里云安全保障研发中台的面试，为啥一题没 A 出来还能约上面啊？

2025-03-19

一身病，约活检项目花了一上午，最近眼睛也是又干又痒。下午还有腾讯的线下面试真的是……我真的有时间把会议论文给写了吗……唉，早知道会这样，还不如早点把会议给写完了，或者索性早点放弃会议论文都行啊。

饿了吗电话约面了，怎么突然面试就扎堆了啊。笑死，我问面试官能不能四月份面，面试官有点难绷。

2025-03-20

虽然之前也有面试过了，但是还是很焦虑…阿里云的面完了，手撕算法堪堪堪撕出来。而且面试问的八股好多啊……不知道这个会议论文我应该怎么处理了，在想要不要和导师摊牌专心准备找实习。

2025-03-21

腾讯企业微信的面试，我去了会不会就是直接当炮灰的。不行，看了牛客网上的面经越看越焦虑啊。明天腾讯 wxg 的面试咋整啊…直到现在我觉得我的项目准备的都不是很好，好焦虑……

才发现我把携程的笔试看漏了，根本没参加……

2025-03-22

今天腾讯的面试还是很简单的，面试官很友善，但是我的简历是后端开发，今天的面试是客户端开发的，整的我有点没绷住

美团的第二次笔试，结果还是这样，只 A 了 1.15

2025-03-23

今天也算是体验到了什么叫一日闲暇了，不说了今天早点去看看能不能理发

腾子的客户端开发过了 emmm 这叫我怎么准备啊？复试？？

2025-03-26

依然不知道要怎么准备，还是老实把八股再背背吧。不过至少有个好消息，会议论文的事导师说了没空就算了。

2025-03-27

携程的笔试算我觉得我做得最好的一次了，四道题 A 三道。可惜的是最后一道有点思路但是没有时间了。第三题动态规划不应该花这么久的……

今天的企业微信客户端二面，感觉还不错，就是问了 C++ 的八股没办法答上来……

2025-03-29

今天淘天的笔试依然是当头一棒啊，这第一题就把我给卡住了。后面发现第二题我的思路是完全正确的。唉，可惜没有如果，太遗憾了（

2025-03-30

蚂蚁的算法题让我有点信心了，线段树写出来过了还是挺爽的，但是第三题排列组合我确实也想不出来应该怎么做比较合适，思路有是有的，但是当排列组合题做怪怪的

2025-03-31

不小心把人面试官的约面电话给挂了，难绷。

我怎么又有一个电话没接到啊？？

没有对比就没有伤害呀，果然同伴压力是很搞人心态的，但是谁让别人有硬实力呢？

2025-04-01

美团今天的手撕代码还是比较顺利的，总体感觉也很不错。

然后就是一天约了四家的面试，要死了。

2025-04-02

饿了吗的面试，表现很差，问项目项目不会，问 Java Java 不会，面试官往后推了半个小时，然后就只面了 30 分钟。笔试也没有，面完就挂了。只能说自己的准备还是不够吧，然后还缺点运气

2025-04-03

上午携程的面试，自我感觉还行，然后算法是 int 和 ip 的转换

下午淘天阿里妈妈，其实面得也还行吧。算法让撕了两道题，但是算法题第二题有点复杂了，说了思路但是没写完就被打断了。秒挂。

2025-04-07

假期结束，中间还是玩得蛮爽的，不过马上又要开始面试了

其实看到小伙伴们已经有 offer 了，心里也蛮不是滋味的…但谁让我技术就不比不上别人呢

停止自我攻击，你的娱乐是有意义的。不要在别人的赛道上疲于奔命。

2025-04-08

腾讯客户端三面，题目简单，没有手撕，但是面试官迟到了。算法那块我居然没想起来用快速选择，害，还是理解不够哇

2025-04-09

字节一面，总体感觉还行

2025-04-10

美团二面，感觉也还不错，没有手撕

2025-04-11

又是连着两个约面电话，要累死了

2025-04-14

腾讯约面委了，妈呀什么时候是个头啊？

字节二面，感觉不错，手撕题想的太困难了实际上很简单。二面过了，但是师兄说生活服务部门不太好。

2025-04-15

携程二面，面得很难受，感觉问题问得很叼钻。更搞笑的是最后面试官一直以为我是校招的……

2025-04-16

字节三面，半个小时就结束了，问得东西也很水，但是问的问题回答上来的也不多。最后手撕题是个二分查找，一开始没注意看题，当成从右上角开始的那题了。

果然，阿里云的二面挂了

2025-04-17

面委面果然还是有点强度的，问得我感觉自己基本都不会

2025-04-20

看了一下美团的进度了，也挂了……不过是白日做梦罢了

2025-04-21

携程怎么 AI 面试完还有口语测评啊？

2025-04-22

蚂蚁拖了这么久，最后简历也是挂了（哪怕笔试做得感觉还行…）

淘天的面试面得我汗流浃背啊……项目问得太细了，不愧是专业的

2025-04-23

淘天挂了，果然不是我能碰瓷的……

2025-04-24

腾讯微信客户端最后也还是挂了，我能说什么呢……有那么一刻我真的在想我到底是哪里有问题了（）

2025-05-06

字节 HR 问入职时间了，我还是比较纠结的，要不要直接去字节还是等携程的 offer

2025-05-14

字节入职

回看一下记录，字里行间不少都是焦虑相关的东西。还有就是流程走一半被挂掉确实会让人 emo 的，尤其是身边同学有第一次面试流程就一次走到尾的，感觉别人总是比我顺利，就会特别 emo。还有腾讯的企业微信客户端，硬控我这么久最后还给我挂了是真的该死啊……

面试之前是真的没有底，对着 Java Guide 反复记忆，面了几场下来，可能大部分面试问题都不算难，包括面试时的手撕算法题也不是特别难，可能我刷题刷得也有点多了（至少比笔试难，但是笔试的算法题即使不太理想也有很大机会进面试的）。关于项目其实我觉得这个项目面试官都是知道你这个是玩具项目（像我同学一样的技术力大佬毕竟是少数），这个项目只是给面试官一个问你问题的方向，具体你的深度大家都清楚，一个实习生能有什么项目经历？但是面试的时候问到项目里的一些技术的时候，我还是有一点捉襟见肘，尤其是问到商城业务相关的内容，以及网络的内容经常不知道面试官想问什么。

面试的时候遇到过很好说话的面试官，在腾讯的线下面试的时候，我可以看出来面试官也不是特别熟练的样子；有的则是阴着一个脸整场面试都很压抑；也有面试迟到的面试官。总之也是见到过各种各样的面试官，也有简历挂一面挂二面挂面委面挂（应该只有腾讯微信有这个，byd 五场面试耗着我是真的气啊……）

大概就是这样吧，我最后去了字节生服，也不知道算不算好。其实做决定的时候只有字节一个有 offer，携程那边磨叽了太久，发了邮件也没个明确回应，我就没等了，入职一个星期携程那才来电话问 offer 的事，那我也只能拒掉了……蚂蚁更是重量级，前几天来约面，算法岗的……反正开发岗都直接给我简历挂了，亏我笔试发挥这么好一点机会都不给的！这个算法估计就是个 kpi，我没什么论文去了也是白给，后面也是直接拒了，面试官问我去哪了，我说我去了字节，真是解气……

再说说字节的 landing 吧，前两周已经开始要做需求了，虽然都是不超过十几行的需求，目前还学不到什么真正的技术，而且字节用的是 Go 所以我感觉我准备 Java 准备了个寂寞。部门里的 mentor 还有同学挺友好的，所以适应得很快。感觉大公司这些用的 CI 工作流什么的很酷！字节是包三餐的，还有好吃的下午茶（可惜要取消了）。至于工作时间感觉比我在学校的时候轻松（草）。

实习的时候，我也和一起实习的同学聊过，想不想转正，或者秋招的时候，还要不要找互联网相关的企业。我也不知道以后会做什么，我可能并不是一个特别擅长规划未来的人，也许我会再考虑一下其他的选择吧……

刚搬到出租屋的时候，床铺也没准备好。就直接买了四件套，把屋子里的席梦思一盖就睡上去了，弹簧硌背啊。后来加了层床垫才舒服一点。又到后面在小红书上刷到有人席梦思放反了睡觉会背疼，我就又检查了一下，草，原来我的也反了啊？翻过来之后是硬的没有弹簧的感觉了，果然舒服了……出租屋的空调和洗衣机也是我清理的，隔应死我了（我觉得我有一点洁癖）。不过也算是勉强安定下来了（但是夏凉被和床单枕套一直没空洗，买到就只把它们晒了一下），自己的生存指南也多了不少东西。

随心而写想到哪里就写到哪里了。总之希望在字节实习顺利（中间可能还要写一篇正刊），祝自己能顺利毕业！也能慢慢找到自己热爱的东西。

2025.07.18

在字节也算是干了俩月了，对实习的新鲜劲儿也过了，就再随便写写自己的想法吧。

感觉想在工作期间写一篇正刊是不太可能了，工作完回屋子，以及周末放假的时候完全没有点开 PDF 看论文的心情。这三个月也对人生以后的路有了一点点的想法，也许从一开始就不用对自己要求那么高，能毕业就行了，两手都抓也许能做到，但可能生活就真的一点意思也没有了……

再一个就是对大厂也没有那么多的滤镜了。没去过其他厂，在就字节而言，觉得自己的工作日常也就那样，每天写写方案，写写需求，查查报警，日子就过去了。不过字节开发的流水线也算是让我这个乡下孩子体验了一把真正的开发流程了（就是一个简单需求要磨叽走完上线流程很烦），让我觉得很酷的那种！

团队里的 mentor 挺友好的，也会适当地指点一下工作方向，不过大多数时间还是靠自己，而且进组也不会有太多的 landing 时间……工作强度吧，我也不知道算不算高，感觉自己还能适应，就是不知道这样长久下去身体受不受得了（作息已经崩坏了，晚上熬夜到两点，早上八点半九点就醒了）。

最需要吐槽的就是住房了，我在上海租房是真的难受啊……中间因为租的房子有问题还搬了一次家，我有点不太想以后在上海工作了，住得不好很影响心情。

以上，近三个月的实习也快要结束了，之后大概就要回学校准备论文（或者毕设），以及秋招了，感觉自己这几个月记的笔记也没多少能 copy 出来……
