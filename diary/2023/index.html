2023 年日记

2023 年日记

封面 ID: 104172322

一月

一月一日

一眨眼一年就过去了，这个博客也建立了大半年了。回想起来，自己能一步步把想法变成现实，还是挺不可思议的。现在，我已经不仅搭建了一个博客，尝试了生成代码文档并部署在 github page，还开了一个专门放笔记的网站，甚至头脑一热整了个服务器，并完成了博客的迁移（虽然最后博客还是挂在了 Github 上，服务器只放了个个人主页）。这一步步都如同翻山越岭一般，即使是现在回头一看，也会觉得做东西太多了，顺带一提，这几天还整了个机器人部署到服务器上了（服务器总算是没闲着了）。某种意义上我的执行力还是蛮强的，乍一看不可能完成的任务，最后居然真的被我完成了！

在每天折腾的过程中，我也学到了很多没用的知识。知识十分散乱，包括前端的三大件，后端服务器，爬虫，Linux 系统基本知识，还有我想不起来的各种知识。虽然这么学习注定无法精深，但我觉得学到这些“没用”的、七零八落的知识，本身还是挺有用的。至少我的眼界能开阔一分，少一分自负，多一分谦虚，能更熟练地学习如何解决问题，也让我对自己的学习能力更有了些信心。（嘿嘿，现在我能熟练地在 Github 部署多个网页并绑在我的各子域名下面了，厉害吧！）

总之，接下来的一年我可能不能像去年一样花这么多时间做自己的事情了，希望接下来的毕业设计我能继续发挥我不怕困难的执行力，并以钻研精深的专业态度顺利完成它！

一月九日

寒假已经过了一半了，还剩下一个月，感觉自己毕业设计还啥也没做，心里很慌。这些时间我也没闲着，每天都在做着自己的事情，学习新知识，为的就是保持自己的专注力。但是，一旦任务转移到毕设上来，我就一点动力也没有了。一方面可能是因为学习新知识是我主动学的，而毕设是个强制的、被动的任务，另一方面可能是因为我一做毕设，就会边做边刷碧蓝航线（我有罪）。接下来的时间怎么办呢，再过两周就要过年了，又要荒废好多时间，寒假就要结束了，必须要想个办法了！肝碧蓝航线也得节制点了，最好其他事也要放放了……

二月

二月五日

Nooooo！怎么就二月份了？？？？毕设毕设毕设啊！

唉，感觉到现在实验还是没有啥实质性的进展。

二月二十一日

第一篇 Obsidian 日记，尝试一下。

这几天入坑 Obsidian ，发现它真的是神仙软件，完美解决我的笔记归档和发布问题。而且个人使用免费，社区插件支持很好，相见恨晚！

这个模板的功能好赞啊，爱了！

之后再继续慢慢探索吧，该睡觉了！

二月二十二日

现在主要的问题是，两台电脑上都有日记要记，生活和科研要分很开吗？

再一个就是 workflowy 定位的问题， 我觉得还是应该留着，是时候想想大纲笔记的优势了。

麻了，今天才发现我的 bot 已经挂了，手机能登录，服务器上登不了。唉，再等两会过了看看吧。

今天参加了同学聚餐，后面他们还歌唱了，我不太擅长，我就当氛围组了 XD

二月二十三日

今天的事情做得还行，没有像前两天一样摸鱼。

二月二十四日

今天比较充实，但是毕设任务进度不仅没推进，反而更难顶了，明天得先把太湖的数据做了（

二月二十五日

二月二十六日

摸了，下午再好好干活。

怨仇我的怨仇，怎么还抽不到啊！？

QuickAdd 真不错啊！

三月十四日

三月十七日
