日常记录

上一次草草写日记，都已经是2021年的事了，到现在也已经过了一年了。刚开始写日记的时间应该是高二的时候了，那时候还用的是纸质的日记本，后来上了大学，才开始用 OneNote 记日记。现在翻一翻以前记的日记，多半是感觉日记幼稚好笑，一小部分阴郁的部分也让我感到难过，甚至让我想起来我到现在依然没有解决的问题，不过也有那么几刻，会让我觉得自己在某一天突然变得明事理起来……总之，希望借着小小的 Blog ，能够找回点当初的感觉吧。虽然悲伤与痛苦更让人记忆深刻，但是生活中的快乐请一定不要忘记啊！

二零二二年

（这里空空的，看来丘卡又偷懒了呢）

二月二十八日

今天再仔细翻看以前的日记时，我又发现了不少的东西。总体来说，高中那段时间的日记是很压抑的，但反倒是最丰富的，等到了大学的时候，时间多了，反而没有高中的时候写得勤了。高中主要内容就是奥赛、学习成绩，一些自己听说的消息。大学时间依然是担心考试成绩，不过多了很多在马圈中的感受。由于是以前的自己的记录，所以隔了这么久，代入感还是很强的。

在这将近三年的日记中，有各种各样的“短日记”，充数日记太多了，随便放几个：

原来每天我都好无聊

2018.9.12

中午不睡，again。

2019.1.19

(写了日期，但没有写任何的东西)

2019.3.2

黑夜从未远去。

2019.9.29

好累呀，就是累。

2020.2.11

最近脖子有点不舒服。

有让我看了觉得很二很尬的不知所云的，太尴了，不想放上来……

还有的，看得让我心碎。看着当时满怀希望的自己，我的心在滴血。就像是从上帝视角看去，明知道眼前的这个傻孩子要受伤，却没办法阻止。对不起那时的自己：

知道的结局的我眼泪掉下来

2018.8.31（国初0+2天）（高考279天）

从容——也无风雨也无晴

明天就要出发了，心情波动却毫无焦虑。褪去高二的稚气，我相信，我，我们大家，一定能秀出最真的自己。

结局如你所见，我在国初就出局了（教练表示遗憾）。我只不过一直在假努力吧……

2019.6.3（高考4天）

也不过就这样嘛，一千天也就这样过去了。

人生强大，舍我其谁。

语文老师临考前几天，送给同学们她订制的书签。我的书签上写的是“人生强大，舍我其谁”。
至于结局……我成了一只 WHUer，已经低于我的心理预期很多了，奇迹只发生在我旁边的人身上。我一直在假努力，奇迹自然不会眷顾我。

2019.6.9

一切的一切都从今天开始。

▓▓▓▓▓，开始，▓▓，你好啊

加黑幕是不想涉及到他，他其实也没做错任何事，只是我完全不能接受这种结果。

高考之前的路是很漫长很痛苦的，我一直都在挣扎。我并不是有天赋的人，竞赛输了以后，上清北的想法就非常非常愚蠢了，追赶文化课的时间真的不多，我每天都在思考我各个学科的问题到底出在哪里……总之心路历程很长很单调，就不放出来了。

有些还比较正经，或者有那么一瞬间让我有了时空错位的感觉：

自我感觉良好

2018.8.25（国初6+2天）（高考285天）

如果不能一鸣惊人，那也应该清清嗓子，一展歌喉对吧！

2019.4.19（高考49天）

是世界运转得太快，还是我走得太慢了？

再把日记往前翻。49天，就像从天上看着无知的过去的怀着愚蠢希望的我。

2019.5.10（高考28天）（冬奥会1000天）

挂了……

说真的，当我翻日记时发现我还记了冬奥会倒时计的时候，那种恍惚感，真的奇妙。（虽然挂科不是什么好事……）

日记里还有一些关于我与我的创造出的角色们一起对话的内容，太过精分，我也不放出来了。

之前写的日记虽然是零零散散的，但是回顾起来还是很花时间的，我只能摘选其中的一点点，作为这次新开始之前的总结（有些太羞耻了，实在不想放上来）。其实这些在现在看起来，真的挺傻挺幼稚的，但是它却帮我挺过了那段最黑暗的时间，重新阅读它，就像找回了不小心弄丢的记忆，也让我能从过去那个屡战屡败屡败屡战的小家伙身上汲取前进的毅力与勇气。

三月六日

写这次日记的时候已经是凌晨0：34了，配置cordova卡了一晚上，挺让人心累的，本来打算今天（3.6之前）把空间信息移动编程的作业搞定的，结果又得拖到明天（3.6）了，明天还得把数字摄影测量的论文写一写，不然到时候编程作业更没时间了。今天难得抽了点时间，写一点感受，以后再搞这个Blog就不知道是什么时候了，希望有空能继续把移动编程记录补上。

很累很累，为了这门课已经花了我接近两天的时间……说实话我其实有点后悔当初没有把这门课给撤掉了，希望最后的结果能够好一点。

三月八日

希望这周末能有时间静下来好好写一篇长点的日记。

现在有些课程的进度已经快过半了，作业难度确实大，赶作业挺累的。卫星摄影测量的编程虽然是小组任务，但是太硬核，我现在碰都不想碰。数字摄影测量的小论文着实恶心到我了，一想到后面还有书面作业，我就头大。而且数字摄影测量马上也要布置编程任务了。近景摄影测量作业阳间，但最近也有一个课间实习需要编程。空间信息移动编程一周一次作业很折磨。地理建模方法和地理监测与应用的大论文还没定，应该也不远了（砸桌）……（果然还是多媒体技术这种外院课最好啊……）

之后有空把自己写的矩阵类还有 Manim 代码给搬过来，还有以前写的坦克大战代码啥的，也分享出来。

三月十一日

我真的是无语了，这老师布置作业怎么还带把 DDL 提前的，真当我们只上他一门课啊？看来这周末要狂赶论文了……

三月二十日

像这样天天宅在寝室里要么是摸鱼要么是写代码，我都不知道有什么好写的东西了。难道说我已经失去了对美好事物的嗅觉了吗，还是说天天沉浸在这么个紧张节奏中对其他事情都提不起兴趣了？

生活中琐碎的事太多了，好事往往都马上忘掉了，难过的事却能一直留到晚上……躺在床上睡不着的时候，常常想起的，也都是一些悲伤的事情。明明有那么多开心的事啊，我的脑子却总喜欢放一些难过的“电影”。有的时候半夜醒来的时候，眼角都还是润润的。也许这是我在寻求我自己的怜悯吧。

四月八日

每次小组作业我都能感觉到我的编程水平有多么低，同样是大学三年，为什么我就没什么长进呢？别人也是该玩就玩，为什么他们效率就这么高呢？看到别人有那么多事情要做还能安排得井井有条，我真的是佩服他们。难道我就只是那种要靠大把时间顶着低效率干活的人吗？

四月二十二日

核酸卡片真不错！




五月三十日

本来打算继续躺平的，最后还是想拽住5月的尾巴写一点点吧。

这周没课没实习，但是忙着写四个实习报告，中间还处理了一下研究生导师的问题，实在是心焦，感觉以后再也没有闲暇时间了啊。

我这人，就喜欢和别人比，比不过还比，最后破防的还是自己。算了……懒得写了，越写越烦……

六月十五日

没有期限的承诺配不上我的信任。现在要准备夏令营了，我也没得心力再去管这些破事了，滚吧，赶快从我面前消失掉吧。

七月十七日

“如愿以偿”地拿到了优营，但是我却并不怎么开心。人总是想往高处走的，但我研究生可能还是要留在武大。我总觉得，和我当初的预期不符，我在一次次地降低要求……

八月一日

今天帮妈妈打杂做饭的时候，我妈妈揶揄我，说我不会做饭，以后他们也享不了口福了。细细思索，好像现在身边的朋友基本上也都不怎么会做菜做饭，以后不说能不能吃上我妈的菜了，就连家常菜恐怕都不太容易吃到了吧？

八月十二日

先是家里老了人，再是科三，这一天真是难捱啊。

生死无常，今天我算是明白了其中的含义了……我本来对姥爷他也没什么感觉的，但是他走得确实挺突然。先是一通电话说是人断气了，然后我妈打电话去问又说根本没这回事，于是我和我妈以为闹了乌龙啥的。结果过了一会，突然又是一通电话实锤了消息，我妈饭也没吃就出家门了……

考科三的时候确实紧张，但是等得时间太长了，也给磨得没有脾气了。虽然中间有几个失误，不过最后好像是满分过的。害，当时听到电脑报合格的时候，我觉得我的暑假终于解脱了（虽然本来就不应该要去练车的）。

八月十七日

我等了一年半的稿子终于要出了。终于，终于要解脱了！只能说，约稿的一大乐趣是收到稿的一瞬间，过后再想想就没有意思了。不过，我以后是不会再去约稿了，毕竟等稿子的过程对我来说太痛苦了。

八月三十一日

某个混蛋终于回来了，带着噩耗喜讯回来的……真是太混蛋了！可恶啊啊啊啊！

假期，就要结束了，真的过得好快！我的暑假啊！

九月一日

要收拾东西了，不知道下次回家会是什么时候了，到时候会不会想念父母呢？

九月二日

这个暑假，我都干了些什么呢？

  • 神农架旅游（大概三四天）

  • LaTex 入门，尝试 LaTex 论文模板（大概一周）

  • 优化博客内容（穿插在暑假中）

  • Lua 学习，并实现简易的俄罗斯方块（大概一周）

  • Python 学习，主要是关于 Python 的最基础知识的拾遗（大概两天）

  • 正则表达式学习（大概一天）

  • 摆烂式看论文（毫无动力可言，完全是在打发时间）

  • 改专利书（大概一天）

  • 学 Hanabi（穿插在暑假中）

  • 运筹学、数学建模、数值分析走马观花（两天）

九月十八日

🤣今天其实也没干成啥事，论文也没开始动，实习报告也没写，就这样摸鱼摸过去了，还不如在寝室里好好玩呢。

九月二十日

培训任务要命了啊……不过培训完了没布置啥作业，不然我的实习报告就真的写不完了。

九月二十一日

仔细想想，自己好像又有两个星期没怎么动过博客了。之前新开的一个文章到现在还没有动笔。看来大四的时间也很紧张啊。

九月二十八日

今天，武汉大学摄影测量与遥感学硕待录取。从此又是各奔前程。希望我的“被迫”坚持，能有所收获吧。

十月八日

😥乐死我了，这就到十月份了啊？我日记都还没写几天呢，怎么就十月了？我不能接受啊！

十月十八日

😫没有目标，我很迷茫。我知道我需要提升自己，但我不知道怎么提升自己。

十月二十四日

论文一个字也憋不出来，好难受！

十一月五日

又是一事无成的一个月啊！但是任务摆在那里，这个月不完成也得完成！写完手里这篇论文就要开始准备毕设了（

十一月六日

😓为什么数据做不出来啊？奶奶滴！为什么匹配不上啊！！！

十一月二十日

😫好家伙，段老师给我的改动也太多了，我是该感动还是该害怕？我论文里真的好多废话，服了。

十一月二十九日

🔊关在宿舍楼里的压抑生活，完全高兴不起来啊…

（未卜先知是不存在的哦）
